# Dicteewebsite

Deze website werkt rechtstreeks met GitHub Pages.

## Online zetten

1. Maak op GitHub een nieuwe repository.
2. Upload alle bestanden en mappen uit deze map.
3. Ga in GitHub naar Settings.
4. Klik links op Pages.
5. Kies bij Source: Deploy from a branch.
6. Kies Branch: main.
7. Kies Folder: /root.
8. Klik op Save.

Na enkele minuten staat de website online.

## Nieuw woordpakket toevoegen

1. Maak een nieuwe map in audio, bijvoorbeeld audio/wp3.
2. Zet daarin de mp3 bestanden.
3. Voeg in data/wordpackages.json een nieuw woordpakket toe.
4. Commit de wijzigingen.

GitHub Pages werkt de website daarna automatisch bij.
