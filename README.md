# Dicteewebsite versie 3.0

Deze versie is gemaakt voor GitHub Pages.

## Uploaden naar GitHub

Upload de volledige inhoud van deze map naar de hoofdmap van je repository.

De hoofdmap moet er zo uitzien:

```text
index.html
css/
js/
data/
audio/
README.md
```

Niet zo:

```text
dicteewebsite_v3_final/
  index.html
```

## Leerkrachtpagina

Open:

```text
https://jufnina.github.io/dictee/?admin=1
```

Vink aan welke woordpakketten leerlingen mogen oefenen en kopieer daarna de leerlinglink.

## Audio toevoegen

Nieuwe woordpakketten komen in:

```text
audio/wp6/
```

De woorden voeg je toe in `data/wordpackages.json`.
